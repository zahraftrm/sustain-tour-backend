# Aplikasi Travel - Sustain Tour

## Deskripsi
Aplikasi travel ini adalah sebuah platform yang memungkinkan pengguna untuk mencari dan memesan paket wisata secara online. Aplikasi ini dirancang untuk memudahkan pengguna dalam merencanakan perjalanan mereka.

## Fitur-Fitur
* **Pencarian Paket Wisata**: Pengguna dapat mencari paket wisata berdasarkan destinasi, tanggal, dan harga.
* **Rekomendasi Destinasi**: Aplikasi ini menyediakan rekomendasi destinasi wisata berdasarkan minat pengguna.
* **Ulasan dan Rating**: Pengguna dapat memberikan ulasan dan rating untuk hotel dan paket wisata yang telah mereka gunakan.

## Tujuan
* Memudahkan pengguna dalam merencanakan perjalanan mereka.
* Menjadi pilihan utama bagi pengguna yang ingin merencanakan perjalanan mereka dengan mudah dan nyaman.
